# apsolution.at

## Overview

This repository contains the full source code of the apsolution.at website.
The project is built using a modern web stack and is structured for scalability, performance and easy maintenance.

## Tech Stack

* React
* Next.js
* Tailwind CSS

## Project Structure

* app → Application routes and pages
* components → Reusable UI components
* lib → Utility functions and logic
* public → Static assets such as images, favicon and sitemap.xml
* styles → Global styles

## Deployment

The website is currently live at:

https://apsolution.at

The project can be deployed on any server that supports Node.js.

## Getting Started

Clone the repository:

git clone https://github.com/APSolution-html

Install dependencies:

npm install

Run the development server:

npm run dev

Build for production:

npm run build

Start production server:

npm start

## Notes

* The repository includes the complete codebase
* All necessary configuration files are included
* Environment variables are not included for security reasons
